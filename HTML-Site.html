Introdução ao HTML5

O HTML (Hypertext Markup Language) é a linguagem fundamental para a criação e estruturação de conteúdo na web. Ele utiliza uma abordagem de "marcação" para definir elementos que compõem uma página, permitindo a apresentação de informações em navegadores da internet.

Funcionalidades do HTML:

  • Estruturação: O HTML fornece tags que permitem estruturar o conteúdo de uma página, como parágrafos (<p>), títulos (<h1>, <h2>, etc.) e listas (<ul>, <ol>).
  • Links e Navegação: Através da tag <a>, é possível criar links para outras páginas, recursos ou locais na mesma página.
  • Multimídia: O HTML5 introduziu suporte nativo para áudio (<audio>) e vídeo (<video>), permitindo a incorporação de elementos multimídia diretamente nas páginas.
  • Formulários: Utilizando tags como <form>, <input>, e outras, o HTML permite a criação de formulários interativos para coleta de dados do usuário.

A evolução do HTML ocorreu ao longo de diversas versões, sendo o HTML5 a mais recente. Cada nova versão trouxe aprimoramentos, novas funcionalidades e melhorias na semântica, proporcionando uma experiência de desenvolvimento mais rica e flexível.

Títulos

Os títulos são importantes para estruturar a hierarquia do conteúdo em uma página. Eles vão de <h1> a <h6>, onde <h1> é o título mais importante e <h6> é o menos importante.

            <h1>Este é um Título de Nível 1</h1>
            <h2>Este é um Título de Nível 2</h2>
            <h3>Este é um Título de Nível 3</h3>

Utilize títulos de forma consistente para melhorar a acessibilidade e a compreensão do conteúdo.

Parágrafos e Quebras de Linha

Os parágrafos são criados usando a tag <p>, e as quebras de linha podem ser feitas com a tag <br>.

Exemplo de parágrafo:

            <p>Este é um parágrafo de exemplo.</p>

Exemplo de quebra de linha:

            Texto antes da quebra<br>Texto após a quebra.

Listas

Listas Ordenadas

Listas ordenadas são criadas usando a tag <ol>, e cada item é representado por <li>.

            <ol>
                <li>Item 1</li>
                <li>Item 2</li>
                <li>Item 3</li>
            </ol>

Listas Não Ordenadas

Listas não ordenadas são criadas usando a tag <ul>, e cada item é representado por <li>.

            <ul>
                <li>Item A</li>
                <li>Item B</li>
                <li>Item C</li>
            </ul>

Links

Você pode criar links usando a tag <a>. O atributo href especifica o destino do link.

            <a href="https://www.exemplo.com">Link de Exemplo</a>

Além disso, você pode adicionar atributos como target="_blank" para abrir o link em uma nova guia.

            <a href="https://www.exemplo.com" target="_blank">Abrir em Nova Guia</a>

Imagens

Para adicionar imagens, use a tag <img>. O atributo src especifica o caminho da imagem, e alt fornece um texto alternativo.

            <img src="caminho/para/sua/imagem.jpg" alt="Descrição da imagem">

Vídeos

Para incorporar vídeos, use a tag <video>. O atributo controls adiciona controles de reprodução.

            <video width="320" height="240" controls>
                <source src="caminho/para/seu/video.mp4" type="video/mp4">
                Seu navegador não suporta o elemento de vídeo.
            </video>

Tabelas

Use as tags <table>, <tr> para linhas, <td> para células e <th> para cabeçalhos de tabela.

            <table border="1">
                <tr>
                    <th>Nome</th>
                    <th>Sobrenome</th>
                </tr>
                <tr>
                    <td>João</td>
                    <td>Silva</td>
                </tr>
                <tr>
                    <td>Maria</td>
                    <td>Souza</td>
                </tr>
            </table>
